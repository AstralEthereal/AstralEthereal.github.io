vlan的划分与ip配置

vlan划分是以端口为单位

  • 一般来说，只要为vlan配置ip,那么这台交换机的管理ip就是该vlan的ip
  • 对于vlan，其实只有两种操作：设置状态，划分vlan。划分的格式都为Switchport access/trunk(端口类型)

划分vlan

进入端口(配置前提) int 端口
配置端口状态 Switchport mode access/trunk
对access口配置vlan Switchport access vlan xxx
对trunk口配置vlan Switchport trunk allowed vlan only/add/remove x,x,x

ip配置

进入vlan(配置vlanip前提) int vlan
配置ip地址 ip address xxx.xxx.xxx.xxx/xx
删除ip地址 no ip address
增删vlan (no) vlan range (使用逗号分隔或-连续)

远程

配置远程登录命令

进入远程登录线路 line vty 0 4
配置密码 password xxx
删除密码 No password
启用密码 login

VTY(Virtual Teletype Terminal，虚拟终端连接)是指用户远程登录交换机所使用的线路。一般有15条，即同时最多支持15个Telnet会话。我们在这配置0-4号，即5条。
